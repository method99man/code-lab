# 🚀 Code Lab

Mój osobisty poligon doświadczalny programowania.
Projekty, eksperymenty i ćwiczenia w JavaScript, Pythonie i technologiach webowych.

## 📌 O projekcie

Code Lab to repozytorium, w którym rozwijam swoje umiejętności programistyczne poprzez:
- budowanie małych aplikacji
- tworzenie mini-projektów do portfolio
- eksperymentowanie z nowymi technologiami
- rozwijanie architektury i czystego kodu

Repo zawiera projekty o różnym poziomie trudności — od podstawowych aplikacji CLI po bardziej rozbudowane dashboardy i gry.

## 🛠️ Technologie
🌐 Frontend
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API

## 🐍 Python
- Python 3.x
- Programowanie obiektowe (OOP)
- Praca z plikami
- API i JSON

## 🚀 Jak uruchomić projekty
### JavaScript
1. Wejdź do folderu projektu
2. Otwórz index.html w przeglądarce lub użyj Live Server.

### Python
``` bash
cd python/nazwa-projektu
python main.py
```
