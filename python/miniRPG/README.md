# 🧙 Mini RPG (Python Terminal Game)

A simple turn-based RPG game created in Python, played entirely in the terminal.

## 🎮 Features

* ⚔️ Turn-based combat system
* 💥 Critical hit mechanic (20% chance)
* 🧠 Basic enemy AI (heals when low HP)
* ❤️ Healing system with max HP limit
* 🎒 Inventory system (potions)
* 🎁 Random loot drops after battle
* 👹 Multiple enemies (Goblin, Orc, Dragon)

## 🛠️ Technologies

* Python 3
* Built-in `random` module

## ▶️ How to run

```bash
python rpg_game.py
```

## 🎯 Gameplay

* Choose actions each turn:

  * Attack
  * Heal
  * Use potion
* Defeat enemies to survive and collect loot
* Game ends when your HP reaches 0
